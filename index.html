Verkefni 1

Í þessu verkefni var ég að búa til þessa síðu og notaðist ég við ChatGPt til að búa til HTML skránna og fínpússuðum við hana saman. Ég vill hafa síðuna stílhreina og einfalda í notkun með eins fáum flipum og ég kemst upp með þar sem mér finnst persónulega ekki skemmtilegt þegar vefsíður eru með of marga takka og sub-categories. Við þetta verkefni fékk ég einnig aðstöð frá vini mínum sem er í hugbúnaðarverkfræði.

Myndir

Hér er pláss til að setja inn myndir ef það þarf í verkefnum.

GitHub – Hvernig ég hlóð upp

git clone https://github.com/Teddi-is/Teddi-is.github.io.git
Klóna GitHub repository-ið yfir á tölvuna mína svo ég geti unnið með skrárnar locally. 
cd .\Teddi-is.github.io
Færi mig inn í möppuna sem var búin til þegar repository-ið var klónað. 
ls
Sýni innihald möppunnar til að sjá hvaða skrár eru til staðar. 
mkdir assets
Bý til sérstaka möppu fyrir myndir og önnur skjöl. 
ni index.html
Bý til aðalskránna sem inniheldur HTML kóðann. 
ni styles.css
Bý til CSS skrá fyrir útlit. 
code .
Opna verkefnið í Visual Studio Code til að vinna áfram með kóðann. 
git status
Sýnir stöðu repository-ins og hvaða skrár hafa verið breyttar, bætt við eða eru tilbúnar til að commita. 
git add .
Bætir öllum breyttum skrám við staging svo þær verði teknar með í næsta commit. 
git commit -m "Update verkefni 1"
Vistar breytingarnar í repository-inu með uppfærslu skilaboðum. 
git push
Sendir commit-ið upp á GitHub þannig að breytingarnar birtist í fjar-repository og á vefsíðunni. 
mkdir assets\project1
Býr til möppu fyrir myndir og gögn sem tengjast Verkefni 1.
